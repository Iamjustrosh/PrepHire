## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or bun

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/your-username/prephire.git
   cd prephire
```

2. **Install dependencies**
```bash
   npm install
```

3. **Start the development server**
```bash
   npm run dev
```

4. **Open your browser**
